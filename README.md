# Multi-Model Tool Wear Monitoring and Early Jamming Detection Using Acoustic and Ultrasonic Signals

This repository contains the complete source code, datasets, and experimental evaluation for our research on data-driven predictive maintenance in Industry 4.0. The study proposes a localized, multi-model framework to monitor the structural health of CNC drilling tools throughout their entire life cycle, utilizing audible (common microphone) and ultrasonic acoustic signatures.

---

## 📌 Project Overview

Tool wear in high-speed CNC machining is a continuous degradation process that culminates in catastrophic failure (drill jamming/breakage). This project introduces a robust monitoring system capable of identifying subtle spectral shifts that precede failure using three core architectures across different sensor configurations:

* [cite_start]**Supervised Learning Baseline:** `XGBoost Classifier`.
* [cite_start]**Static Semi-Supervised Learning:** `MLP Autoencoder` (MLP-AE).
* [cite_start]**Temporal Semi-Supervised Learning:** `LSTM Autoencoder` (LSTM-AE).

The core framework evaluates these models under three distinct physical setups to assess environmental noise resilience:
1.  [cite_start]**Audible Channel (Common Microphone):** Captures standard acoustic emissions.
2.  [cite_start]**Ultrasonic Channel:** Purifies the signal by targeting high-frequency bands.
3.  [cite_start]**Hybrid Fusion:** Combines multi-sensor inputs for enhanced localization.

---

## 🔒 Methodology & Local Calibration

To prevent data leakage across different tools and account for the physical amplitude variations inherent to each manufacturing unit, the pipeline incorporates **Leave-One-Drill-Out (LODO)** cross-validation. 

Furthermore, a **Localized Calibration** strategy is applied: the normalization (`StandardScaler`) and anomaly thresholds are computed strictly using the first 5 healthy holes of each individual drill unit. This localized tuning allows the framework to judge a tool against its own baseline, effectively mitigating early-stage false alarms.
